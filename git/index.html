Git Cheatsheet by Swapnil Sanghvi

    Setting Up the Environment

    Git Configure : Set Email & Name of Git User

    git config --global user.name "Your Name"
    git config --global user.email "Your Email"

    Command Line Tool


    Print Working Directory

    pwd

    List Items

    ls

    to view detailed information of list items

    ls -l

    to view hidden files and folder in list items

    ls -a

    to view hidden files and folder with detailed information in list items

    ls -al
    ls -la

    Change Directory : to change directory

    cd ./directoryname

    go to parent directory

    cd ..

    go to home directory

    cd
    ~

    go to root folder

    cd /

    Create Folder

    mkdir foldername

    Create Multiple Folder

    mkdir folder1 folder2 folder3 folder4

    Create folder inside folder

    mkdir -p parentfolder/childfolder

    Create files

    touch file.html

    Create Multiple files

    touch file.txt file.html file.js

    Remove file

    rm file.txt

    Remove file inside a folder

    rm folder1/file.html

    Remove folder : empty

    rmdir folder2

    Remove folder : which is not empty

    rm -R folder3

    Copy File

    cp file.txt folder1/file.txt

    Copy File with Rename

    cp file.html folder1/index.html

    Copy Folder

    cp -R folder1 folder2

    Move File/Folder

    mv file.txt folder1/file.txt

    Rename File/Folder

    mv file.html folder1/index.html

    Tracking Projects


    initialize Git

    git init

    To track files and folder

    git status

    Stage any specific file 

    git add index.html

    Stage all files & Folder

    git add .

    Making Commits 

    git commit -m "Your Commit Message"

    Viewing the commiting history 

    git log
    git log --oneline"
    git log --oneline --graph --all"

    Deleting the git project 

    rm -rf .git

    Additional Git Commands


    Skipping the staging area

    git commit -am "Your Commit Message"
    git commit -a -m "Your Commit Message"

    Track the changes that have not been staged

    git diff

    display the changes of already staged files

    git diff --staged

    Remove Files in Git

    git rm file.txt

    Remove Files Forecelfully in Git

    git rm -f file.html

    Move and Rename Files in Git

    git mv home.html index.html
    git mv index.html folder1/index.html

    Untrack an already tracked file

    git rm --cached file.txt

    Unstage an staged file

    git restore --staged file.txt

    Take file from last commit

    git checkout --file.txt

    Rollback to the previous commit, lossing all modified files

    git checkout -f

    Setup alias in git

    git config --global alias.shortcut command

    Non linear Development Branching


    Create a new branch

    git branch branchname

    View all branch

    git branch

    View commit of all branch

    git branch -v

    Moving between branch

    git checkout branchname

    Creating and Checkout to a new branch

    git checkout -b branchname

    Deleting Merged Branch

    git branch -d branchname

    Deleting a non-merged Branch

    git branch -D branchname

    Non linear Development Merging and Rebasing


    Merge Branch

    git checkout master
    git merge branchname

    Rebase Branch

    git checkout branchname
    git rebase master

    Git and Github


    Adding Remote in git

    git remote add remotename github-repo-url
    git remote add origin https://github.com/SwapnilSanghvi/Git-Cheatsheet.git

    view all Remote

    git remote

    view all Remote with location

    git remote -v

    Push commits in github

    git push remotename branchname
    git push origin master

    Push commits in github

    git remote set-url remotename github-repo-url
    git remote set-url origin https://github.com/SwapnilSanghvi/Git-Cheatsheet.git

    Remove Remote

    git remote remove remotename
    git remote remove origin

    Fetch changes from github

    git fetch

    Merging changes of fetch

    git merge

    Git Pull = Git Fetch + Git Merge

    git pull remotename branchname
    git pull origin master

    Miscellaneous Git Tools


    Show me the commit changes

    git show commitid

    Stash : to save changes without doing commit

    git add .
    git stash

    Show Stash

    git stash list

    Revert Stash

    git stash apply stashid

    Remove Stash

    git stash drop stashid

    Cleaning working repository

    git clean -f -d

    Change last commit and its content

    git commit --amend

    Rewriting History


    Travelling to the past commits

    git checkout commitid

    Switch back to master branch

    git switch -

    Making new branches from old commits

    git switch -c branchname

    Revert commit - undo the changes of commit

    git revert commitid

    Reset commit - delete all commits from the given commit id


    if you use git reset --soft then you can't undo the commits but you have a chance to recover files with the help of git add .
    but in git reset --hard you can't recover anything

    git reset commitid
    git reset --soft commitid
    git reset --hard commitid

    Push forecefully : deleting commits from github

    git push -f origin master

    Cherry-Picking Commits : pick the changes from a specific commit from any branch and merge with another branch

    git cherry-pick commitid
