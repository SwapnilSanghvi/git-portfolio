Jquery Cheatsheet By Swapnil Sanghvi

    jQuery Selectors : in jquery we have only one way to select html element

    $(selecter) //Syntax

$('h1') //Selects all h1 tags
$('h1:first') //Selects only first h1 tag
$('h1').first() //selects only first h1 tag
$('h1:first-of-type') //Selects only first h1 tag
$('h1:nth-of-type(2)') //Selects second h1 tag
$('h1:last') //Selects only last h1 tag
$('h1').last() //selects only last h1 tag

    Manipulating HTML Elements: we can manipulate by using css() method.

    $(selector).css(property,value) //Syntax

$('h1').css('color','white');
$('h1').css('background','red');
$('h1').css('border','5px solid green');

    jQuery text() Method: We can use this method to get and set text of the matched elements. i.e this method acts as both getter and setter method.

    $('h1').text() //get
$('h1').text('Swapnil Sanghvi') //set

    jQuery html() Method : We can use this method to get and set html content.

    $('h1').html() //get
$('h1').html('Heading H2')

    jQuery attr() Method : We can use this method to get and set attribute values.

    $('img').attr('src') //get
$('img').css({width:'150px',height:'150px'}); //set

    jQuery val() Method : We can use val() method to get value of the first matched element.

    $('input').val() //get
$('input').val('swapnil') //set

    jQuery addClass() Method : We an use this method to add specified class/classes to the set of matched elements.

    $('li').addClass('high')

    jQuery removeClass() Method : Remove a single class/classes from each matched element.

    $('li').removeClass('high')

    jQuery toggleClass() Method : We can use this method to add and remove classes from the matched elements.

    $('li').toggleClass('low') //It will add 'low' class for every li tag
$('li').toggleClass('low') // It will remove 'low' class because this class is already set

    jQuery click() Event : jQuery click() method can be used to add a click listener to the elements.

    $('h1').click(function(){
//Whenever click event happend, this function will execute
});

    jQuery keypress() Event : whenever we are pressing the key if we want to do any activity automatically then we should go for keypress() method.

    $('input').keypress(function(){
})

    jQuery on() Event : this is smiliar of javascript addEventListner Function 

    $('h1').on('mouseover',function(){
//Whenever mouseover event happend, this function will execute
})

$('h1').on('mouseout',function(){
//Whenever mouseout event happend, this function will execute
})

$('h1').on('click',function(){
//Whenever click event happend, this function will execute
})

$('h1').on('dblclick',function(){
//Whenever double event happend, this function will execute
})

    jQuery Fading Effects : jQuery defines the following methods for fading purposes.

    //Hide the matched elements by fading them to transparent.
$(selector).fadeOut(time_in_milisecond,function);

// Display the matched elements which are fadeout
$(selector).fadeIn(time_in_milisecond,function);

//  Display or hide the matched elements
$(selector).fadeToggle(time_in_milisecond,function);

    jQuery Sliding Effects : jQuery defines the following methods for sliding effects purposes.

    // Hide the matched elements with a sliding motion.
$(selector).slideUp(time_in_milisecond,function);

//  Display the matched elements with a sliding motion.
$(selector).slideDown(time_in_milisecond,function);

// Display or hide the matched elements with a sliding motion.
$(selector).slideToggle(time_in_milisecond,function);
