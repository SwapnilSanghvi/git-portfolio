Markdown Cheatsheet By Swapnil Sanghvi

    Headings : To create a heading, add number signs (#) in front of a word or phrase

    # h1 Heading
## h2 heading
### h3 heading
#### h4 heading
##### h5 heading
###### h6 heading

    Paragraph : To create paragraphs, use a blank line to separate one or more lines of text.

    This paragraph is writtern in markdown

i love markdown.

    Line Breaks : To create a line break or new line (<br>), end a line with two or more spaces, and then type return.

    first way to break the line.
second way to break this line.<br>
example of line break.

    Bold Text : To bold text, add two asterisks without spaces around the letters.

    **bold text**.

    Italic Text : To italic text, add one asterisks without spaces around the letters.

    *italic text*.

    Bold and Italic : To emphasize text with bold and italics at the same time, add three asterisks or underscores before and after a word or phrase.

    ***bold and text***.

    Subscript : To create a subscript, use one tilde symbol (~) before and after the characters.

    H~2~O

    Superscript : To create a superscript, use one caret symbol (^) before and after the characters.

    X^2^.

    Strikethrough : To strikethrough words, use two tilde symbols (~~) before and after the words.

    ~~Strikethrough.~~

    Highlight : To highlight words, use two equal signs (==) before and after the words.

    I am ==Swapnil Sanghvi==.

    Blockquotes : To create a blockquote, add a > in front of a paragraph.

    > Blockquote.

    Nested Blockquotes : Blockquotes can be nested. Add a >> in front of the paragraph you want to nest. 

    >> Nested blockquote.

    Ordered Lists : To create an ordered list, add line items with numbers followed by periods.

    1. item 1
2. item 2
3. item 3

    Unrdered Lists : To create an unordered list, add dashes (-) in front of line items.

    - item 1
- item 2
- item 3

    Code : To denote a word or phrase as code, enclose it in backticks (`).

    `<html> This is html code </html>`

    Code Blocks : To create code blocks, indent every line of the block by at least four spaces or one tab.. 

        <html>
    <head>
    </head>
    </html>

    Horizontal Rules : To create a horizontal rule, use three or more asterisks (***), dashes (---). 

    This is a Horizontal Line

---

Another one Horizontal Line

***

    Links : To create a link, enclose the link text in brackets (e.g., [Markdown Cheatsheet]) and then follow it immediately with the URL in parentheses (e.g., (url)).

    [Link](url).

    Links Title : You can optionally add a title for a link. This will appear as a tooltip when the user hovers over the link. To add a title, enclose it in quotation marks after the URL.

    [Link](url "title").

    URLs : To quickly turn a URL or email address into a link, enclose it in angle brackets.

    <url>

    Formatting the Link : The reference-style link is formatted with two sets of brackets. 

    Content [Link][1]

[1]: <url> "title"

    Images : To add an image, add an exclamation mark (!), followed by alt text in brackets, and the path or URL to the image asset in parentheses.

    ![alt-text!](imageurl "title")

    Table : To add a table, use three or more hyphens (---) to create each column’s header, and use pipes (|) to separate each column. For compatibility, you should also add a pipe on either end of the row.

    | TableHeader | TableHeader |
| ----------- | ----------- |
| Tabledata   | Tabledata   |
| Tabledata   | Tabledata   |

    Table Alignment : You can align text in the columns to the left, right, or center by adding a colon (:) to the left, right, or on both side of the hyphens within the header row.

    | Table Header | Table Header | Table Header |
| :---         | :---:        | ---:         |
| left-align   | center-align | right-align  |

    Fenced Code Blocks with Syntax Highlighting : The basic Markdown syntax allows you to create code blocks by adding three backticks (```) and To add syntax highlighting, specify a language next to the backticks before the fenced code block.

    ```html
<html>
    <head>
        <title>Show and highlight code in markdown</title>
    <head>
    <body>
    </body>
</html>
```

    Task Lists : To create a task list, add dashes (-) and brackets with a space ([ ]) in front of task list items. To select a checkbox, add an x in between the brackets ([x]).

    - [x] Design Web UI
- [ ] Design Front End
- [ ] Back end and Deploy

    Automatic URL Linking : That means if you type http://www.example.com, your Markdown processor will automatically turn it into a link even though you haven’t used brackets..

    http://www.example.com

    Disabling Automatic URL Linking : If you don’t want a URL to be automatically linked, you can remove the link by denoting the URL as code with backticks.

    `http://www.example.com`
