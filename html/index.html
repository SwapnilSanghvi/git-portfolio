    Regular Expression - Notes

    <!-- Regular Expression
[0-9] -- Any Digit
[a-z] -- only lower case alphabet symbol
[A-Z] -- only upper case alphabet symbol
[a-zA-Z] -- any alphabet symbol
[a-zA-Z0-9] -- any aplphanumeric character
[abc] -- either a or b or c
[^abc] --Expect a or b or c any other character
[^a-zA-Z0-9] --Any Character Expect alpha numeric character
a* -- any number of a-s includin zero
a+ --atleast one a

[6-9] [0-9] [0-9] [0-9] [0-9] [0-9] [0-9] [0-9] [0-9] [0-9] = [6-9] [0-9] {9}
                -->

    Regular Expression - Examples

    <form action="thankyou.html" method="get">
  <h3>Case-1: The value should contain minimum 5 character and maximum 10 character</h3>
  <label for="case1">Case1</label>
  <input type="text" name="case1" pattern=".{5,10}" title="enter minimun 5 and maximun 10 character"><br>

  <h3>Case-2: The value should contain onlye digits, exactly 10 digits only allowed</h3>
  <label for="case2">Case2</label>
  <input type="text" name="case2" pattern="[0-9]{10}" title="invalid message"><br>

  <h3>Case-3: The value should contain onlye digits, exactly 10 digits only allowed but first digital should be either 6 or 7 or 8 or 9</h3>
  <label for="case3">Case3</label>
  <input type="text" name="case3" pattern="[6-9][0-9]{9}" title="invalid message"><br>

  <h3>Case-4: password atleast 8 character must be required. the first character should be upper case alphabet symbol remaining any alpha numberic.</h3>
  <label for="case4">Case3</label>
  <input type="text" name="case4" pattern="[A-Z][a-zA-Z0-9]{7}[a-zA-Z0-9]*" title="invalid message"><br>

  <h3>Case-5: first letter uppercase and remaining 7 letter is small and last character is special symbol</h3>
  <label for="case5">Case5</label>
      lt;input type="text" name="case5" pattern="[A-Z][a-z]{7} [^a-zA-Z0-9]" title="invalid message"><br>
  <input type="submit" value="submit">
</form>
