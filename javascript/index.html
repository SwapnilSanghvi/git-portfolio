    var favNumber=[1,4,3,7,8,6]

    unshift() Array Method - To add element at beginning of the array

    var favNumber=[1,4,3,7,8,6]

    var favNumber=[1,4,3,7,8,6]

    indexOf() Array Method - To return index of first occurrence of the given element.

    var favNumber=[1,4,3,7,8,6]

    slice() Array Method - to get part of the array from begin index to end index. slice() doesn't changes the original array and returns array object.

    var favNumber=[1,4,3,7,8,6]

console.log(favNumber)//[1, 4, 3, 7, 8, 6]

    splice() Array Method - splice() changes the original array and returns array object.

    var favNumber=[1,4,3,7,8,6,4,5]
console.log(favNumber.splice(1,5));//[4, 3, 7, 8, 6]
console.log(favNumber)//[1, 4, 5]

    Immutability vs Mutability: Mutability means changeable where as Immutable means Non-Changeable.

    Mutability: Once we creates an array object,we are allowed to change its content. Hence arrays are Mutable. 

    var numbers=[10,20,30,40]
numbers[0]=777
console.log(numbers)//[777,20,30,40]

    Immutability: Once we creates string object,we are not allowed to change the content.If we are trying to change with those changes a new object will be created and we cannot change content of existing object. 

    var name='Sunny'
name[0]='B'
console.log(name)// Sunny
